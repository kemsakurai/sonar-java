共有コーディング規則により、チームは効率的にコラボレーションすることができます。 このルールは、すべての定数名が指定された正規表現と一致するかどうかを検証します。

違反コード例

デフォルトで提供される正規表現 ^[A-Z][A-Z0-9]*(_[A-Z0-9]+)*$:

public class MyClass {
  public static final int first = 1;
}

public enum MyEnum {
  first;
}

準拠方法

public class MyClass {
  public static final int FIRST = 1;
}

public enum MyEnum {
  FIRST;
}
