ラベルが宣言されているがプログラムで使用されていない場合、デッドコードと見なすことができるため、削除する必要があります。

開発者はこのラベルが何に使用されるのか意識しないため、保守性が向上します。

違反コード例

void foo() {
  outer: //ラベルは使用されません。
  for(int i = 0; i<10; i++) {
    break;
  }
}

準拠方法

void foo() {
  for(int i = 0; i<10; i++) {
    break;
  }
}

参考

  • CERT, MSC12-C. - プログラムに対して作用しないコードを検出して削除する
